Mas, O que é "Ciência de Dados" ?

Com a chegada da Era da Informação a quantidade de Informações, de Dados, se tornaram imensas e impossíveis de se administrar no papel, para coletar e organizar todos esses dados só com o auxílio da tecnologia. A Ciência de Dados ou Data Science é o uso de diversas tecnologias, para coletar, armazenar e processar informações, ela se mostrou fundamental para qualquer empresa, para, por exemplo, traçar o perfil de possíveis clientes com base em dados coletados, e a partir daí, criar possíveis estratégias de negócio, de marketing, etc. A Ciência de Dados une alguns ramos, como por exemplo, Ciência da Computação, Inteligência Artifical, para lidar com esses Dados e organizá-los devidamente, é necessário um bom conhecimento em Matemática e Estatística.
